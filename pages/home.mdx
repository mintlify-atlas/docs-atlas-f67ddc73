# Home Page

The home page (`src/app/page.tsx`) is the main landing page of the Mario Rodriguez Portfolio. It showcases skills, featured projects, tech stack, and a call-to-action section.

## Page Structure

The home page consists of several key sections:

1. **Hero Section** - Introduction with availability badge and CTA buttons
2. **Specialties Section** - Three core competencies (Frontend, Backend, DevOps)
3. **Projects Section** - Featured projects grid with preview cards
4. **Process Section** - Four-step workflow visualization
5. **Tech Stack Section** - Technology badges with icons
6. **CTA Section** - Contact call-to-action with WhatsApp and email links

## Projects Section

The featured projects section displays a grid of project cards with two display types:

### Standard Project Card (Web Apps)

```tsx
<Link
  href={`/projects/${project.slug}`}
  className="relative block h-full overflow-hidden rounded-xl shadow-lg border border-transparent group-hover:border-emerald-500/30 transition-colors"
  aria-label={`View details for ${project.title}`}
>
  {cover && (
    <Image
      src={cover.src}
      alt={cover.alt}
      fill
      sizes="(max-width: 768px) 100vw, (max-width: 1024px) 50vw, 33vw"
      className="object-cover transition-transform duration-500 group-hover:scale-110"
    />
  )}

  {/* Hover overlay */}
  <div className="absolute inset-0 bg-emerald-500/20 opacity-0 group-hover:opacity-100 transition-opacity flex items-center justify-center">
    <span className="bg-white text-emerald-600 px-4 py-2 rounded-lg font-bold text-sm shadow-xl hover:bg-slate-50">
      View Details
    </span>
  </div>
</Link>
```

### API Project Card

For projects with `displayType === "api"`, the page uses the `ApiPreviewCard` component:

```tsx
{project.displayType === "api" && project.apiPreview ? (
  <Link href={`/projects/${project.slug}`} className="block h-full">
    <ApiPreviewCard
      className="h-full"
      endpoints={project.apiPreview.endpoints}
      baseUrl={project.apiPreview.baseUrl}
      maxEndpoints={4}
    />
  </Link>
) : (
  // Standard image preview
)}
```

### Fixed Card Heights

All project cards maintain consistent height:

```tsx
<div className="mb-4 h-[220px]">
  {/* Preview area with fixed height */}
</div>
```
