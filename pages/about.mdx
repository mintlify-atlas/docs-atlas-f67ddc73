# About Page

The about page (`src/app/about/page.tsx`) provides information about Mario Rodriguez's experience, technical skills, and professional approach to web development.

## Page Structure

The about page consists of three main sections:

1. **Bio Section** - Personal introduction and development philosophy
2. **Tech Stack Section** - Categorized list of technologies
3. **Approach Section** - Core development principles

## Metadata Configuration

```tsx
export const metadata: Metadata = {
    title: "ABOUT",
    description:
        "Learn more about my experience, technical skills, and professional approach to web development.",
};
```

The metadata is exported for Next.js App Router to generate proper page titles and meta tags.

## Tech Stack Section

Displays technologies organized by category in a responsive grid:

```tsx
const technologies = [
    { category: "Frontend", items: ["React", "Next.js", "TypeScript", "Tailwind CSS"] },
    { category: "Backend", items: ["Node.js", "Python", "PostgreSQL", "MongoDB"] },
    { category: "Tools", items: ["Git", "Docker", "Vercel", "Figma"] },
];
```

### Section Implementation

```tsx
<section className="mb-16">
    <h2 className="text-sm font-medium tracking-widest text-slate-500 mb-8">
        TECH STACK
    </h2>
    <div className="grid sm:grid-cols-3 gap-6">
        {technologies.map((tech) => (
            <div key={tech.category} className="p-6 bg-slate-900/50 border border-white/10 rounded-2xl">
                <h3 className="font-semibold mb-4 text-cyan-400">{tech.category}</h3>
                <ul className="space-y-2">
                    {tech.items.map((item) => (
                        <li key={item} className="text-slate-400 flex items-center gap-2">
                            <span className="w-1 h-1 rounded-full bg-slate-600" />
                            {item}
                        </li>
                    ))}
                </ul>
            </div>
        ))}
    </div>
</section>
```

### Card Features:
- **Glass morphism effect** with `bg-slate-900/50`
- **Category headers** in cyan accent color
- **Bullet points** using small circular spans
- **3-column grid** on desktop, single column on mobile

## Reveal Animations

All sections use the `Reveal` component for scroll animations:

```tsx
<Reveal>
    <header className="mb-12">
        <h1 className="text-3xl sm:text-4xl font-bold tracking-tight mb-4 text-slate-100">
            ABOUT
        </h1>
    </header>
</Reveal>

<Reveal delay={0.1}>
    {/* Bio section */}
</Reveal>

<Reveal delay={0.2}>
    {/* Tech Stack section */}
</Reveal>

<Reveal delay={0.3}>
    {/* Approach section */}
</Reveal>
```

### Delay Pattern:
- **Header**: No delay (0s)
- **Bio**: 0.1s delay
- **Tech Stack**: 0.2s delay
- **Approach**: 0.3s delay

This creates a cascading reveal effect as the user scrolls.
