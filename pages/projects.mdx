# Projects Listing Page

The projects listing page (`src/app/projects/page.tsx`) displays all portfolio projects in a responsive grid with interactive cards and animations.

## Page Type

This is a **client component** as indicated by the `"use client"` directive:

```tsx
"use client";
```

This is required because the page uses:
- Framer Motion animations
- Interactive hover effects
- Client-side state management

## Page Structure

The page consists of two main parts:

1. **Header Section** - Page title and description
2. **Projects Grid** - Grid of project cards with animations


## Project Card Component

Each project is wrapped in a `Link` component with Framer Motion animations:

```tsx
<Link href={`/projects/${project.slug}`} className="block group">
    <motion.div
        whileHover={{ y: -6 }}
        transition={{ type: "spring", stiffness: 420, damping: 28 }}
        className="overflow-hidden bg-slate-900/50 border border-white/10 rounded-2xl hover:border-cyan-400/30 transition-colors"
    >
        {/* Card content */}
    </motion.div>
</Link>
```

### Animation Properties:

- **whileHover**: `{ y: -6 }` - Lifts card 6px on hover
- **transition**: `{ type: "spring", stiffness: 420, damping: 28 }` - Spring animation with specific physics
- **Border transition**: `border-white/10` → `border-cyan-400/30` on hover

## Conditional Preview Display

The card displays different content based on project type:

### API Projects

For projects with `displayType === "api"`:

```tsx
{project.displayType === "api" && project.apiPreview ? (
    <div className="p-4">
        <ApiPreviewCard
            endpoints={project.apiPreview.endpoints}
            baseUrl={project.apiPreview.baseUrl}
            maxEndpoints={4}
        />
    </div>
) : (
    // Standard image preview
)}
```

### Web App Projects

For standard web applications:

```tsx
<div className="relative aspect-video overflow-hidden">
    {project.images?.desktop[0] && (
        <Image
            src={project.images.desktop[0].src}
            alt={project.images.desktop[0].alt}
            fill
            sizes="(max-width: 640px) 100vw, 50vw"
            className="object-cover transition-transform duration-500 group-hover:scale-105"
        />
    )}
</div>
```

### Image Properties:
- **aspect-video**: Maintains 16:9 aspect ratio
- **fill**: Next.js Image component fills parent container
- **sizes**: Responsive image sizing for optimal performance
  - Mobile (≤640px): 100vw
  - Desktop: 50vw (half viewport width)
- **group-hover:scale-105**: Image scales 105% on card hover

Each card includes project metadata:


### Features:

1. **API Badge**: Conditional badge for API projects
2. **Title**: Changes to cyan on hover
3. **Description**: Limited to 2 lines with `line-clamp-2`
4. **Tech Stack**: Shows first 4 technologies using `slice(0, 4)`

## Staggered Reveal Animations

Each project card has a staggered reveal animation:

```tsx
{projects.map((project, index) => (
    <Reveal key={project.slug} delay={0.1 * index}>
        <Link href={`/projects/${project.slug}`} className="block group">
            {/* Card content */}
        </Link>
    </Reveal>
))}
```

### Delay Calculation:
- **First project**: 0.1 * 0 = 0s
- **Second project**: 0.1 * 1 = 0.1s
- **Third project**: 0.1 * 2 = 0.2s
- **Fourth project**: 0.1 * 3 = 0.3s
- And so on...

This creates a waterfall animation effect.
