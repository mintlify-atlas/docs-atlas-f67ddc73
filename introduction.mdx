# Mario Rodríguez Portfolio

A production-ready portfolio website showcasing full-stack development expertise through real-world projects. Built with Next.js 16, TypeScript, and Tailwind CSS, featuring motion animations, responsive design, and optimized performance.

## Overview

This portfolio demonstrates expertise in building fast, scalable web applications with clean UI and robust APIs. The site features:

- **Modern Next.js 16** with App Router and React 19
- **Responsive Design** with Tailwind CSS 4 and custom animations
- **Performance Optimized** with Vercel Speed Insights and image optimization
- **Type-Safe** development with TypeScript 5
- **Motion Animations** using the motion library (motion/react)
- **Contact Integration** with Resend for email functionality

## Key Features

### Portfolio Showcase

Features three production projects with detailed case studies:

- **Booking Platform**: .NET 10 Blazor Server rental booking system with 3-step flow
- **Payroll System**: Enterprise .NET 8 MVC system with PDF generation and legal compliance
- **Inventory API**: Multi-tenant REST API with Clean Architecture and ledger-based tracking

### Interactive UI Components

```tsx title="src/components/ApiPreviewCard.tsx"
// Displays API endpoints with syntax highlighting
<ApiPreviewCard
  endpoints={[
    { method: "GET", path: "/items", label: "List inventory items" },
    { method: "POST", path: "/items", label: "Create item" }
  ]}
  baseUrl="/v1"
  maxEndpoints={4}

```

- **ApiPreviewCard**: Interactive endpoint display for API projects
- **ImageCarousel**: Touch-enabled gallery with navigation
- **PhoneFrame**: Mobile mockup with realistic device frame
- **CodeBlock**: Syntax-highlighted code samples
- **Reveal**: Scroll-triggered animations using motion/react

### Performance Features

- **Image Optimization**: AVIF/WebP formats with Vercel Blob Storage
- **Speed Insights**: Real-time performance monitoring
- **Font Optimization**: Space Grotesk with Google Fonts optimization
- **Dark Mode**: Built-in dark theme with consistent design system

## Portfolio Structure

Projects are defined in a type-safe data structure:

```typescript title="src/data/projects.ts"
export type Project = {
  slug: string;
  title: string;
  description: string;
  stack: string[];
  highlights: string[];
  images?: {
    desktop: ProjectImage[];
    mobile?: ProjectImage;
  };
  repo?: string;
  live?: string;
  displayType?: "webapp" | "api";
  
  // Extended fields for detailed view
  status?: "completed" | "in-progress" | "maintenance";
  stackDetails?: TechCategory[];
  architecture?: string[];
  challenges?: Challenge[];
  
  // API-specific fields
  apiPreview?: ApiPreview;
  capabilities?: string[];
  sampleRequest?: SampleRequest;
};
```
